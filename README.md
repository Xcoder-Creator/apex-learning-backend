This is the backend for the apex learning web app. Check out the frontend here -> [Apex Learning Frontend](https://github.com/Xcoder-Creator/apex-learning-frontend).

## How to Run
1. Clone the repository to your local machine.
2. Open the folder in VSCode.
3. Enter the terminal and run `npm start`.

## Tech Stack
- Node.JS and Express
- MySQL
- XAMPP
- Multer for file upload
